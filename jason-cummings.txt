    1. What is Semantic HTML? A style of marking up html that adds context and meaning to what would otherwise just be strictly text markups.
    2. What is HTML used for? To markup internet pages with content; to give structure to web pages.
    3. What is an attribute and where do we put it? An attribute is a characteristic of an html element, and usually directly follows the element.
    4. What is the h1 tag used for? How many times should I use it on a page? H1 is the main title or headline on a page and should only be used one time per page.
    5. Name two tags that have required attributes: the anchor <a> tag and the image <image> tags both require attributes to function properly.
    6. What do we put in the head of our HTML document? Additional information that isn't contained or displayed in the body of the HTML document, such as charset, page title, etc.
    7. What is an id? A unique identifier (used ONLY ONCE per page) attached to an HTML element that allows you to target only that id for modification.
    8. What elements can I add an id to? All elements can be supplemented with an ID attribute.
    9. How many times can I use the same id on a page? Only once per page.
    10. What is a class? It is an identifier tag, same as an id, except it can be used multiple times on the same page.
    11. What elements can I add a class to? All elements.
    12. How many times can I use the same class on a page? As many as necessary/convenient.
    13. How do I get my link to open in a new tab? By adding the "_blank" attribute to the associated anchor element.
    14. What is the alt attribute in the image tag used for? To give a written description of the image in case the image does not show up, or for the visually impaired.
    15. How do I reference an id? To reference an html ID in CSS, you must put the # symbol before the name of the id that you created.
